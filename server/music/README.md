# music

Сюда кладите свои аудиофайлы (`*.mp3`) **локально**. Они **не** попадают в git
(см. `.gitignore` в корне) — у музыкальных треков есть авторское право, в публичный
репозиторий их класть нельзя.

Плагин играет файлы отсюда: сервер отдаёт их по `/music/<имя>.mp3`, а в плеере
имя прописано в `plugin/ui.html` → массив `TRACKS` (поле `file`).

Чтобы добавить свой трек: положи `mytrack.mp3` сюда и добавь
`{ name: 'Название', file: 'mytrack.mp3' }` в `TRACKS`.
